This page collects all **APIs** the paiOS engine exposes. Use it to find the right interface for your use case (local [IPC](/glossary/#i), remote control, or tool integration).

## Available APIs

| API | Transport | Use case | Reference |
|-----|-----------|----------|-----------|
| **[gRPC API](/reference/api/)** | [UDS](/glossary/#u) (local) or TCP/TLS (remote) | Primary interface: lifecycle, config, status, inference. Type-safe [Protocol Buffers](https://protobuf.dev/). | [gRPC API Reference](/reference/api/): services and messages |
| **REST API** | HTTP | Ollama and OpenAI client compatibility (convenience). | [API domain](/architecture/modules/api/): gateway and routing |
| **MCP Server** | SSE / stdio | Host PC integration (Claude Desktop, OpenClaw, Cursor). Engine exposes tools and sensors. | [API domain](/architecture/modules/api/): MCP Server adapter; [C4 Architecture](/architecture/c4-architecture/): system context |

For **architecture** (routing matrix, port isolation, security), see the [API domain](/architecture/modules/api/) (api) and [Security](/architecture/security/).
