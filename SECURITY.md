## Reporting a Vulnerability

If you discover a security vulnerability in paiOS, please **do not** open a public issue. Instead, report it privately:

- **Email**: [info@aurintex.com](mailto:info@aurintex.com)
- **GitHub**: Use [Private Vulnerability Reporting](https://github.com/aurintex/pai-os/security/advisories/new)
